# backend

Internet Shop backend API.

Core domains:
- shop settings (theme, blocks, branding)
- catalog (products, stock)
- checkout (orders, payment state)
- sale module (listing, offers, ownership transfer)
- audit + RBAC + anti-fraud signals

## Initial API groups

- `/api/shop/*`
- `/api/shop/public/*`
- `/api/shop/sale/*`
