# Product Scope (MVP)

Model:
- single-brand internet shop (one owner)

MVP pillars:
1. Beautiful and usable storefront (catalog, cart, checkout)
2. Trend customization (theme + blocks + branding)
3. Sell store as asset (listing + offer + approval transfer)

Boundary:
- Product logic lives in `INTERNET_SHOP`.
- `OMNIXIUS` stays hub-only with links and ecosystem map.
